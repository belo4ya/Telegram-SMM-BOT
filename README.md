# Telegram SMM BOT
### Что умеет бот:
- Создание поста с картинкой
- Создание поста с внешними ссылками под постом
- Отложенный постинг
- Постинг по расписанию
- Просмотр процесса выполнения Таска
- Заморозка/запуск/удаление Тасков
- Архивирование Тасков
